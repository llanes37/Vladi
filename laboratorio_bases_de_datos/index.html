Ayuda del laboratorio

Que hace cada boton, como empezar y cual es el flujo mas util para practicar SQL aqui.

1. La forma mas rapida de empezar

  1. Si tu profesor te da un `.sql`, pulsa Cargar script SQL.
  2. Si te da una base ya hecha, pulsa Abrir base SQLite.
  3. Si no tienes nada aun, pulsa Cargar demo para probar el entorno.
  4. Cuando la carga termine, las tablas apareceran en la columna izquierda.
  5. Pulsa una tabla del lateral para que el sistema te escriba una consulta base en el editor.

2. Que hace cada boton importante

Nueva base vacia

Borra la base actual y deja el laboratorio limpio para construir una desde cero.

Cargar script SQL

Lee un archivo `.sql` y crea una base nueva con sus tablas y datos.

Pegar script SQL

Sirve para pegar `CREATE TABLE`, `INSERT` o cualquier script sin necesidad de archivo.

Ejecutar script sobre base actual

Anade nuevas tablas o datos encima de la base que ya tienes abierta.

Abrir base SQLite

Abre un archivo `.db`, `.sqlite` o similar sin convertirlo antes.

Resetear

Vuelve al estado original del ultimo archivo o script cargado.

Borrar todo

Vuelve a una base vacia para empezar desde cero.

Ver esquema

Muestra las tablas y sus relaciones para entender la estructura de la base.

Ejecutar seleccion

Ejecuta solo la parte marcada del editor. Muy util si tienes varias consultas.

Ejecutar editor

Ejecuta todo el contenido del editor tal como esta escrito.

3. Flujo recomendado para un alumno

  1. Carga la base del ejercicio.
  2. Comprueba que las tablas aparezcan a la izquierda.
  3. Pulsa una tabla para ver datos reales y entender sus columnas.
  4. Escribe tu consulta en el editor.
  5. Lee el panel de resultados: ahi salen tablas, errores y diagnostico.
  6. Si rompes algo, usa Resetear.
